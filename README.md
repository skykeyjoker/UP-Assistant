UP主小助手，监控并定期导出视频数据。

提供交互友好型用户界面。（有图形界面，没图形界面的为什么不用Python？）

支持自定义查询数据频率。

## 如何使用

进入release界面下载最新版本，解压后即可运行。

可编辑目录下的*config.json*文件，设置`timeout`字段，以设置查询频率（以分钟为单位，默认10min）。
